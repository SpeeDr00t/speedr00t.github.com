Two mechanisms may be used to obtain the RAS password stored in the aVirt interface:
1) Use Snadboy's Revelation tool http://www.#no_exploit/templates/tools_category.html?category=9&path=%20recovery%20%20passwords%20 to view the cleartext password, or
2) Select the 'Properties' tab of the aVirt product. Select 'Internet Connection'. Modify the 'username' entry with a bogus username. Click the 'Test' button. aVirt will attempt to establish the RAS connection. Upon failure to connect (due to the incorrect username) the user will be presented with a popup window displaying the cleartext password stored for the RAS connection.
