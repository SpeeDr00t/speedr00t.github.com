Attackers can use a browser to exploit these issues. To exploit the cross-site scripting issues, an attacker must entice an unsuspecting user to follow a malicious URI.

The following example URIs are available:

http://www.example.com/wp-admin/admin.php?page=forum-server/fs-admin/fs-admin.php&amp;amp;vasthtml_action=structure&amp;amp;do=addforum&amp;amp;groupid=%27%3E%3Cscript%3Ealert%281%29%3C/script%3E
