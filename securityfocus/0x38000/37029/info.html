PHD Help Desk Multiple Cross Site Scripting Vulnerabilities

An attacker can exploit these issues by enticing an unsuspecting victim to follow a malicious URI.

The following example URIs are available:

http://www.example.com/area.php/code
http://www.example.com/area.php?pagina=code
http://www.example.com/area.php?sentido=code
http://www.example.com/area.php?q_registros=code
http://www.example.com/area.php?orden=code
http://www.example.com/solic_display.php?pagina=1&amp;q_registros=code&amp;orden=seq_solicitud_id
http://www.example.com/area_list.php/code
http://www.example.com/area_list.php?pagina=1&amp;q_registros=0code&amp;orden=nombre
http://www.example.com/atributo.php/code
http://www.example.com/atributo_list.php?pagina=1code&amp;q_registros=15&amp;orden=activo&amp;sentido
http://www.example.com/atributo_list.php?pagina=1&amp;q_registros=15code&amp;orden=activo&amp;sentido
http://www.example.com/atributo_list.php?pagina=1&amp;q_registros=15&amp;orden=activocode&amp;sentido
http://www.example.com/atributo_list.php?pagina=1&amp;q_registros=15&amp;orden=activo&amp;sentidocode
http://www.example.com/caso_insert.php/code
