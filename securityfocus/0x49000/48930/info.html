An attacker can exploit this issue by enticing an unsuspecting victim to following a malicious URI.

The following example URI is available:

http://www.example.com/sitecore/shell/default.aspx?xmlcontrol=Application&amp;url=http://www.example.com&amp;ch=WindowChrome&amp;ic=Applications%2f32x32%2fabout.png&amp;he=About+Sitecore&amp;ma=0&amp;mi=0&amp;re=0
