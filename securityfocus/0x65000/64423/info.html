Attackers can exploit these issues by enticing an unsuspecting user into following a malicious URI.

The following example URIs are available:

http://example.com/index.php POST key=XSS&amp; blog=XSS
http://example.com/bmc/wysiwyg/?form=newpost&amp;box=XSS
http://example.com/bmc/wysiwyg/?form=XSS
http://example.com/user.php?action=new_post&amp;blog=XSS
