The following steps illustrate this issue:

1. Go to VPN--&gt;Certificates--&gt; CRL page and load a CRL list.
2. Save the entire configuration.
3. Reboot the TMS zl Module.
4. Once the TMS zl Module is available, go to VPN--&gt; Certificates--&gt; CRL page and the CRL is no longer available.
